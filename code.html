Merkle Tree Root

A program that computes the Merkle tree root corresponding to a set of transactions using SHA-256.

Implementation

  • Language: Golang
  • Version: 1.16 or later

Requirements

  • The program must output the Merkle root.
  • SHA-256 must be used to perform the hashing at each step, you may use a library for this.
  • The code must be tested and documented.

Usage

Input

  • A file named transactions.txt in the current directory.
  • The file should contain hexadecimal transaction hashes, one per line.

Output

  • The Merkle root in hexadecimal format.

Installation

  1. Clone the repository:
    2. git clone https://github.com/YOUR-USERNAME/merkle-tree-root.git
  2. Build the program:
    3. cd merkle-tree-root go build

Testing

A script that generates a random set of transactions, computes the Merkle root using both the implemented code and a reference implementation, and verifies that the results match.

./test.sh

Documentation

The code is documented using GoDoc comments.
